Welcome to the Cameroon QWERTYAZERTY Keyboard for Keyman. This page will explain how to type in the vernacular languages of Cameroon using the Cameroon QWERTYAZERTY Keyboard.

The Cameroon QWERTYAZERTY Keyboard, one of a set of virtual keyboards maintained by the Language Technology team of SIL Cameroon, allows you to type any languages that follow the General Alphabet of Cameroonian Languages. The Ethnologue1 lists 279 living languages in Cameroon, and 275 of these are indigenous. For a list of these languages, view Cameroon in the Ethnologue.

3 Combinations

As shown above, each key on the Cameroon Keyboard can be used to produce multiple characters (letters).

  1. If you tap the E key once, you will get a lowercase "e".
    Sometimes you may need to use two keys to get the letter you want. The SHIFT key is a key on a computer keyboard. It is used to type capital letters (like this: "ABC") instead of lowercase ("abc").
  2. If you hold the Shift key and tap the E key once, you will get a capital "E".
    The Cameroon Keyboard is special, and it is possible to type other Characters not printed on your keyboard using the "CAM" (Cameroon) key. On your Cameroon QWERTYAZERTY Keyboard, the CAM key is the semicoloneclamation point key (;!). Each time you need the CAM key, just press ;!. To get the special characters shown on the right side of each key in the chart (like this: “əɔɓ”) instead of common characters (like “eob”), you need to use the CAM key.
  3. If you tap the ;! key once, and then press the E key, you will get schwa "ə".
  4. If you tap the ;! key once, then hold Shift and type E, you will get capital schwa “Ə”.
    In review, if you look at the keyboard chart, you’ll see that most keys on the Cameroon keyboard can be used to produce up to four different characters in this way. Use the chart to find the letters you need for your language, and use a combination of Shift and the CAM Key (;!) to type them.
  5. Unlike French, Cameroonian diacritics (accents) are typed after the letter. For example, you would type: ;! e 'ù to get "ə̀"

4 Specific Keyboard Combination Details:

Computers bought in different countries tend to have different physical keyboard layouts. When using virtual keyboards, it is MUCH easier to learn the keyboard if the keys are in the same place as the letters you want to type.

The Cameroon QWERTYAZERTY keyboard is designed for computers with the following physical keyboards:

  • US English (QWERTY)
  • UK Engish (QWERTY)
  • Swiss French (QWERTZ)
  • French (France)
  • Arabic 102
Note: If your computer has an QWERTY or QWERTZan AZERTY Keyboard or something similar, consider removing Cameroon QWERTYAZERTY and try the Cameroon QWERTYAZERTY Keyboard

4 Physical Keyboards

If you choose to connect a physical keyboard (i.e. through Bluetooth or USB), the physical keyboard will respond like a desktop keyboard. See the Physical Keyboard Section for combinations.
Note: If your physical keyboard uses an AZERTY (French) keyboard layout, consider switching to the Cameroon AZERTY Keyboard.

5 Switching to Non-Keyman Keyboards

You can switch between several keyboards. On most phones, the process is:

  1. Tap in a text area to make sure the keyboard is showing.
  2. Swipe from the top of the screen down to the bottom.
  3. Choose the button 'Select Keyboard.
  4. Choose the desired keyboard.
