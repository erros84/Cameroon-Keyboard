Welcome to the Cameroon QWERTYAZERTY Keyboard for Keyman. This page will explain how to type in the vernacular languages of Cameroon using the Cameroon QWERTYAZERTY Keyboard.

Bienvenue au Clavier camerounais QWERTYAZERTY pour Keyman. Cette page explique comment saisir du texte dans les langues vernaculaires du Cameroun à l'aide du clavier QWERTYAZERTY du Cameroun.

The Cameroon QWERTYAZERTY Keyboard, one of a set of virtual keyboards maintained by the Language Technology team of SIL Cameroon, allows you to type any languages that follow the General Alphabet of Cameroonian Languages. The Ethnologue lists 279 living languages in Cameroon, and 275 of these are indigenous. For a list of these languages, view Cameroon in the Ethnologue.

Le clavier camerounais QWERTYAZERTY, un des claviers virtuels maintenus par l'équipe de Technologie Linguistique de SIL Cameroun, vous permet de saisir toutes les langues qui suivent l'Alphabet Général des Langues Camerounaises. L'Ethnologue recense 279 langues vivantes au Cameroun, dont 275 sont autochtones. Pour une liste de ces langues, verrez Cameroun dans l'Ethnologue.

3 Combinations

3 Comment faire les combinaisons de touche

As shown above, each key on the Cameroon Keyboard can be used to produce multiple characters (letters).

Chaque touche du clavier Camerounais peut vous servir à produire des caractères (lettres) différents.

  1. If you tap the E key once, you will get a lowercase "e".
    Sometimes you may need to use two keys to get the letter you want. The SHIFT key is a key on a computer keyboard. It is used to type capital letters (like this: "ABC") instead of lowercase ("abc").
  2. Si vous tapez la touche e une fois, vous obtiendrez un « e » en minuscule.
  3. If you hold the Shift key and tap the E key once, you will get a capital "E".
    The Cameroon Keyboard is special, and it is possible to type other Characters not printed on your keyboard using the "CAM" (Cameroon) key. On your Cameroon QWERTYAZERTY Keyboard, the CAM key is the semicolonexclamation point key (;!). Each time you need the CAM key, just press ;!. To get the special characters shown on the right side of each key in the chart (like this: "əɔɓ ") instead of common characters (like "eob"), you need to use the CAM key.
  4. Parfois, il faut utiliser deux touches sur le clavier pour obtenir la lettre souhaitée. La touche MAJ (SHIFT) est une touche sur le clavier d’un ordinateur. Elle est utilisée pour saisir les lettres en majuscule (par exemple : « ABC ») au lieu de lettres en minuscule (« abc »). Si vous appuyez la touche Maj et tapez la touche E une fois, vous obtiendrez l'« E » en majuscule.
  5. If you tap the ;! key once, and then press the E key, you will get schwa "ə".
  6. Le Clavier Camerounais est spécial, et il est possible de saisir d’autres caractères non imprimés sur votre clavier à l’aide de la touche « CAM » (Cameroun). Sur votre Clavier Camerounais QWERTYAZERTY, la touche CAM est le point-virgulepoint d' exclamation (;!). Chaque fois que vous avez besoin de la touche CAM, tapez la touche ;!. Pour obtenir les caractères spéciaux situés sur le côté droit de chaque touche dans la disposition du clavier (par exemple : « əɔɓ ») au lieu de caractères simples (comme « eob »), vous devez utiliser la touche CAM. Si vous tapez la touche ;! une fois, puis tapez la touche E, vous obtiendrez le schwa « ə ».
  7. If you tap the ;! key once, then hold Shift and type E, you will get capital schwa “Ə”.
    In review, if you look at the keyboard chart, you'll see that most keys on the Cameroon keyboard can be used to produce up to four different characters in this way. Use the chart to find the letters you need for your language, and use a combination of Shift and the CAM Key (;!) to type them.
  8. Parfois, il faut utiliser trois touches pour obtenir le caractère souhaité. Si vous tapez la touche ;! une fois, puis appuyez sur le Shift en tapant le E, vous obtiendrez le schwa en majuscule « Ə ».
  9. Unlike French, Cameroonian diacritics (accents) are typed after the letter. For example, you would type: ;! e 'ù to get "ə̀"

4 Specific Keyboard Combination Details:

4 Combinaisons de touche spécifiques:

Computers bought in different countries tend to have different physical keyboard layouts. When using virtual keyboards, it is MUCH easier to learn the keyboard if the keys are in the same place as the letters you want to type.

Les ordinateurs achetés dans différents pays ont tendance à avoir des claviers physiques différents. Lorsque vous utilisez des claviers virtuels, il est BEAUCOUP plus facile d'apprendre le clavier si les touches sont à la même place que les lettres que vous voulez taper.

The Cameroon QWERTYAZERTY keyboard is designed for computers with the following physical keyboards:

Le clavier Cameroun QWERTYAZERTY est conçu pour les ordinateurs équipés des claviers physiques suivants :

  • US English (QWERTY)
  • UK Engish (QWERTY)
  • Swiss French (QWERTZ)
  • German (QWERTZ)
  • Arabic 101 (Qwerty)
  • Anglais États-Unis (QWERTY)
  • Anglais Royaume Unis (QWERTY)
  • Allemand (QWERTZ)
  • Arabe 101 (QWERTY)
  • French (France)
  • Arabic 102
  • Français Français (AZERTY)
  • Arabe 102 (AZERTY)
Note: If your computer has a QWERTY or QWERTZan AZERTY Keyboard or something similar, consider removing Cameroon QWERTYAZERTY and try the Cameroon QWERTYAZERTY Keyboard
Note: Si votre ordinateur a un clavier an QWERTY or QWERTZ AZERTY ou quelque-chose de similaire, Envisagez de retirer le clavier Cameroun QWERTYAZERTY et installez le clavier Cameroon QWERTYAZERTY.
